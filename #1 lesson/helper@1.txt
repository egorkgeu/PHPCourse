git init - инициализация проекта.
    Внутри папка создается скрытая папка .git.
    Внутри папки .git лежит конфигурацинные файлы.
    При помощи этих конфигурационных файлах идет связь между локальным репозиторием и удаленным.
    Также внутри этого конфигурационного хранится история git

если в PhpStorm файлы подсвечиваются красным - это значит,
что данные файлы не отслеживаются git, не попадают в коммит и при отправлении файлов
на удаленный репозиторий эти файлы не отправятся.

git status
    показывает, какие изменения произошли с веткой.

Суть git - это хранить историю изменений файлов.
git сохраняет состояние файла на текущий момент.

git add
    Чтобы гит начал отслеживать файл, ему об этом нужно сказать.
    И делается это при помощи данной команды
    Варианты использования:
    1. git add . - добавить все неотслеживаемые файлы
    2. git add {{КАКОЙ_НИБУДЬ_ФАЙЛ}} - он добавит данный файл в git
    3. При помощи PhpStorm

git remote add origin {{ССЫЛКА_НА_УДАЛЕННЫЙ_РЕПОЗИТОРИЙ}}

git push origin {{НАЗВАНИЕ_УДАЛЕННОЙ_ВЕТКИ}}

git branch
    Показывает список веток, текущую ветку и удаляет ветки

Функция(Действие)
    Она работает только с теми параметрами, которые в нее передаются
    В своей области видимости
