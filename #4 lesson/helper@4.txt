1. git checkout -b lesson#4
    Флаг {{-b}} создает новую ветку
    {{-b}}  - {{--branch}}
    Варианты использования:
    git checkout -b {{НАЗВАНИЕ_ВЕТКИ}}
    git checkout --branch {{НАЗВАНИЕ_ВЕТКИ}}

    Флаг {{-b}} создает ветку, а команда checkout переключается на нее
    То же самое, если по очередно выполнить следующие команды:
    git branch -c {{НАЗВАНИЕ_ВЕТКИ}}
    git checkout {{НАЗВАНИЕ_ВЕТКИ}}
